Sparse matrix multiplication.

This function multiplies a sparse matrix with a dense matrix. The sparse matrix has, for each row, all of its non-zero values at the beginning of the row and the rest are zeros. The number of zeroes is stored in the first element of each row. Example : 2/5 X X 0 0 0 1/5 X 0 0 0 0 3/5 X X X 0 0
