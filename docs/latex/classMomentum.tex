\doxysubsection{Detailed Description}
\doxylink{classMomentum}{Momentum} optimiser. 

This class implements the \doxylink{classMomentum}{Momentum} optimisation algorithm for updating the weights and biases of a neural network. It uses a moving average of the gradients to update the weights and biases.
